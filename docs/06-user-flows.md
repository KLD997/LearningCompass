# Parent and Student User Flows

## Parent Flow (MVP)

1. Create account and sign in.
2. Add one or more student profiles.
3. Set per-subject level placements.
4. Choose curriculum source per subject.
5. Build first week plan in assignment planner.
6. Student completes lessons.
7. Parent reviews completion, quiz scores, and notes.
8. Parent adjusts pacing, assigns remediation/advance tasks.

## Student Flow (MVP)

1. Student signs in.
2. Lands on Today dashboard.
3. Clicks Continue on next lesson.
4. Completes hosted lesson OR launches external lesson.
5. Submits quiz / marks offline activity complete.
6. Sees progress indicator and badge feedback.
7. Returns to queue for next lesson.

## Exception Flows

### Missing work handling
- Parent dashboard flags overdue assignments.
- Parent can reassign, mark optional, or skip with note.

### External lesson unavailable
- Student sees fallback prompt to notify parent.
- Parent replaces with alternative lesson.

### Younger learner completion
- Parent confirms completion and adds observational note.
